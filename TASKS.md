# TASKS

## 현재 목표

후쿠오카 여행 웹앱의 간단한 UI 개선 테스트를 진행한다.

## 작업 범위

### 1. 일정 카드 UI 개선

* 일정 카드의 모서리를 조금 더 둥글게 변경한다.
* 카드에 은은한 그림자를 추가한다.
* 카드 간 간격을 조금 더 넓혀서 모바일에서 보기 편하게 만든다.

### 2. 버튼 UI 개선

* 주요 버튼에 hover 효과를 추가한다.
* 모바일 화면에서 버튼 높이와 터치 영역을 조금 키운다.
* 버튼 텍스트가 너무 작으면 가독성 있게 조정한다.

### 3. 모바일 화면 확인

* 390px 너비 기준으로 화면이 깨지지 않는지 확인한다.
* 일정 카드, 버튼, 제목 영역이 너무 붙어 있지 않게 여백을 조정한다.

## 작업 방식

수정하기 전에 먼저 다음 내용을 설명한다.

1. 어떤 파일을 수정할지
2. 왜 그 파일을 수정하는지
3. 예상되는 변경 결과

그다음 실제 코드를 수정한다.

## 완료 조건

* 기존 기능은 그대로 유지한다.
* UI만 가볍게 개선한다.
* 수정 후 빌드가 성공해야 한다.
* 가능하면 GitHub에 커밋한다.
