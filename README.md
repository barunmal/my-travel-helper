# 여행 도우미

여행 일정, 구매리스트, 지도, 체크리스트, 예산을 모바일에서 보기 쉽게 관리하는 웹앱입니다.

## 실행

```powershell
npm.cmd install
npm.cmd run dev
```

## 빌드

```powershell
npm.cmd run build
```

## 문서

`docs/` 폴더는 유지보수와 여행 데이터 정리를 위한 작업 공간입니다.

- `docs/TASKS.md`: 해야 할 일과 진행 상태
- `docs/IDEAS.md`: 기능 아이디어와 개선안
- `docs/TRAVEL_PLAN.md`: 여행 일정과 장소 정보
- `docs/SHOPPING.md`: 구매리스트와 쇼핑 메모
- `docs/CHANGELOG.md`: 변경 내역
